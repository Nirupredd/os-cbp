About This Project

Project Overview

The Disk Scheduling Algorithms Visualization project is an educational tool designed to help students and professionals understand the various disk scheduling algorithms used in operating systems. This interactive web application allows users to visualize how different algorithms handle disk access requests, providing insights into their efficiency and behavior.

Features

Comprehensive Wiki

Detailed explanations of each disk scheduling algorithm, including their principles, advantages, and limitations.

Interactive Simulation

Visualize how each algorithm works with custom input parameters and see the results in real-time.

Algorithm Comparison

Compare the performance of different algorithms side by side to understand their relative efficiency.

Step-by-Step Animation

Watch animations that demonstrate how each algorithm processes disk requests sequentially.

Algorithms Covered

  • FCFS (First Come First Serve) - The simplest approach that processes requests in the order they arrive.
  • SSTF (Shortest Seek Time First) - Selects the request that requires the least disk arm movement from its current position.
  • SCAN - The disk arm moves in one direction servicing requests until it reaches the end, then reverses direction.
  • C-SCAN (Circular SCAN) - A modified version of SCAN that provides more uniform service by treating the disk as a circular list.
  • LOOK - Similar to SCAN but the arm only goes as far as the last request in each direction.
  • C-LOOK (Circular LOOK) - A modified version of LOOK that provides more uniform service by treating the disk as a circular list.

Technical Implementation

This project is built using modern web technologies:

  • HTML5/CSS3 - For structure and styling
  • JavaScript - For interactive functionality and algorithm implementation
  • Bootstrap - For responsive design and layout
  • Plotly.js - For data visualization and graphing
  • jQuery - For DOM manipulation and event handling

Educational Purpose

This tool is designed primarily for educational purposes to help computer science students, operating system enthusiasts, and professionals visualize and understand disk scheduling concepts. It can be used as a supplementary learning resource in operating systems courses or for self-study.
